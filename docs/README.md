# Machine Learning Models

A collection of machine learning models for various tasks.

## Features

- Pre-trained models for common tasks
- Easy-to-use APIs for training and inference
- Support for multiple frameworks (TensorFlow, PyTorch)
- Comprehensive documentation

## Installation

```bash
pip install machine-learning-models
```

## Quick Start

```python
from ml_models import Classifier

model = Classifier.load("pretrained/resnet50")
predictions = model.predict("input.jpg")
```

## Documentation

Full documentation is available at [https://ml-models-docs.org](https://ml-models-docs.org).
